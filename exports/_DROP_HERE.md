# Cross-Project Export Drop Zone

Drop status cards here from other projects when you need the Magnum Architect's input.

## Status Card Format

```markdown
## Export From: [Project/Chat Name]
**Date:** YYYY-MM-DD
**Topic:** One-line summary

### What happened:
(2-3 sentences)

### Decision needed:
(What you want the Architect to weigh in on)

### Context the Architect needs:
(Anything relevant they wouldn't already know)
```

Copy that template, fill it in, paste it as a new file in this folder named `YYYY-MM-DD-topic.md`.
